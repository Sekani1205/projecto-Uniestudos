Regimes de Capitalização: Juros Simples

No Regime de Capitalização Simples, a taxa de juros incide diretamente sobre o valor do capital. Em cada período, o juro é obtido pelo produto do capital inicial pela taxa unitária. Desta forma, os juros são iguais em cada período. É também chamado de Juros Simples.

De uma forma simples podemos dizer que, o o regime de juros será simples quando a taxa de juros incidir apenas sobre o valor inicial, ou seja, sobre juros obtidos em cada período não incidirão novos juros.

Esta fórmula é básica tanto para o cálculo dos juros como dos outros valores financeiros mediante simples dedução algébrica:

Exemplo 1

Um capital de $ 80.000,00 é aplicado à taxa de 2,5% ao mês durante um trimestre. Pede-se determinar o valor dos juros acumulados neste período.

Solução:
c = $ 80.000,00
i = 2,5% ao mês (0,025)
n = 3 meses
J = ?

J = C . i . n
J = 80.000,00 X 0,025 X 3
J = $ 6.000,00

Exemplo 2:

Um negociante tomou um empréstimo pagando uma taxa de juros simples de 6% ao mês durante nove meses. Ao final deste período, calculou em $ 270.000,00 o total dos juros incorridos na operação. Determinar o valor do empréstimo.

Solução:
C = ?
i = 6% ao mês (0,06)
n = 9 meses
J = $ 270.000,00

C = J / i . n
C = 270.000 / 0,06 . 9
C = 270.000 / 0,54
= $ 500.000,00

Exemplo 3

Um capital de $ 40.000,00 foi aplicado num fundo de poupança por 11 meses, produzindo um rendimento financeiro de $ 9.680,00. Pede-se para apurar a taxa de juros oferecida por esta operação.

Solução
c = $ 40.000,00
i = ?
n = 11 meses
J = $ 9.680,00

i = J / C . i
i = 9.680 / 40.000 x 11
i = 9.680 / 440.000
i = 0,022 x 100
i = 2,2%

Exemplo 4

Urna aplicação de $ 250.000,00, rendendo urna taxa de juros de 1,8% ao mês produz, ao final de determinado período, juros no valor de $27.000,00. Calcular o prazo da aplicação.

Solução
c = $ 250.000,00
i = 1,8% ao mês (0,018)
n = ?
J = $ 27.000,00

n = J / C x i
n = 27.000 / 250.000 x 0,018
n = 27.000 / 4.500
n = 6 meses

Taxas de Juros Proporcionais

As taxas proporcionais são utilizadas no regime de juros simples e reportam-se a períodos diferentes e a relação entre elas é igual a relação que existe com os períodos a que se reportam.


Basicamente quando falamos sobre taxas proporcionais necessáriamente estamos falando sobre o regime de juros simples. Aqui ocorre uma relação de proporcionalidade.


Para fazer a conversão das taxas trabalhamos com multiplicação ou divisão de acordo o caso. Quando partimos de uma unidade de tempo menor para uma unidade de tempo maior, usamos a multiplicação, quando passamos de uma unidade de tempo maior para uma menor, usamos a divisão.

Exemplo 1:

Calcule a taxa mensal proporcional à taxa anual de 24%.

Solução

i´ = ? a.m.

i´ = i / n
i´ = 0,24 / 12 i´ = 0,02 x 100
i´ = 2% a.m.

Exemplo 1

Uma pessoa aplica$ 18.000,00 à taxa de 1,5% ao mês durante 8 meses. Determinar o valor acumulado ao final deste período.

Solução
c = $ 18.000,00 i = 1,5% ao mês (0,015) n = 8 meses M=?

M = C (1 + i . n)
M = 18.000,00 (1 + 0,015 x 8)
M = 18.000,00 x 1,12 = $ 20.160,00

Exemplo 2

Uma dívida de$ 900.000,00 irá vencer em 4 meses. O credor está oferecendo um desconto de 7% ao mês caso o devedor deseje antecipar o pagamento para hoje Calcular o valor que o devedor pagaria caso antecipasse a liquidação da dívida.

Solução
M = $900.000,00
n = 4 meses
i = 7% ao mês (0,07)
C=?

C = M / (1 + i . n)
C = 900.000 / (1 + 0,07 x 4)
C = 900.000 / 1,28
C = 703.125,00
